# 🧠 Sistema de Recomendações de Rotina Saudável

Projeto em Python que registra hábitos do usuário, analisa a rotina diária e gera recomendações inteligentes baseadas em regras.

## 🚀 Funcionalidades
- Cadastro de múltiplos usuários
- Registro de hábitos diários (sono, água, exercício, humor)
- Recomendações automáticas de rotina saudável
- Relatórios de progresso do usuário
- Validação de dados de entrada
- Interface em terminal (CLI)
- Testes automatizados com Pytest
- Banco de dados SQLite

## 🧠 Inteligência do Sistema
O sistema analisa os hábitos registrados e gera recomendações como:
- Dormir mais cedo se o sono for insuficiente
- Beber mais água se a hidratação for baixa
- Sugerir exercícios físicos
- Recomendar pausas ou meditação em dias de estresse
